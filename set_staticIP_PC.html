Set a static IP on your PC

  1. using the IP range determined in step 3 (the first 3 sets of numbers are the same, however the last set must be unique. i.e. 192.168.1.XXX)
  2. Go to the windows start menu
  3. Type “Control Panel” and select the control panel
  4. Select “Network and Internet”
  5. Select “Network and Sharing Center”
  6. Select “Change adaptor settings”
  7. Right click “Ethernet (may also be LAN or Local Area Connection)” and select “Properties”
  8. Select “Internet Protocol Version 4” so that it is highlighted and then select “Properties” in the lower right portion of the window.
  9. Change the settings to “Use the Following IP address”
  10. Set a static IP on your PC using the IP range determined in step 3 (the first 3 sets of numbers are the same, however the last set must be unique. i.e. 192.168.1.XXX)
  11. Ensure the “Subnet Mask” shows 255.255.255.0
  12. Enter the Default gateway from step 3 for the addresses of “Default Gateway” and “Preferred DNS”. Note that these are identical and “alternate DNS is not required
  13. Select “OK”
