Welcome to the IP tool

  1. Directly connect your PC to the Main repeater via ethernet cable.
  2. Turn off your WIFI on your PC.
  3. Open the "Start" menue
  4. control panel
  5. Select "View by category"
  6. Select "System and Security"
  7. Select "Windows Firewall"
  8. Select "Advanced Settings"
  9. Select "Windows Firewall Properties"
  10. On "Domain Profile" tab, set "Firewalls state" dropdown to "Off"
  11. On "Private Profile" tab, set "Firewalls state" dropdown to "Off"
  12. On "Public Profile" tab, set "Firewalls state" dropdown to "Off"
  13. Select "Apply"
  14. Select "OK"
  15. Minimize the "Windows Firewall with Advanced Security" window. We will want to turn these back on later for your PC's protection.
