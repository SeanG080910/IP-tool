Identify the native network IP scheme

  1. Directly connect your PC to the router on site
  2. Go to the windows start menu
  3. Type “CMD” to open windows command prompt
  4. Type “ipconfig” and hit the “enter” key
  5. Verify and record the default gateway (i.e. 192.168.1.1)
  6. The first 3 sets of numbers are the network IP range (i.e. 192.168.1.1)
