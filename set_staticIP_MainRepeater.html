+Set a static IP for the Main Repeater

  1. Open RA2 configuration software (Essentials/Inclusive)(NOTE: If your program is already open, close and reopen the program)
  2. Open your project file
  3. Find Main repeater in the design screen
  4. Disable DHCP
  5. Input a UNIQUE static IP using the IP range determined in step 3 (the first 3 sets of numbers are the same, however the last set must be unique. i.e. 192.168.1.XXX)
  6. Click “Show Advanced” and ensure that the “Subnet mask”, “Gateway Address” & “Preferred DNS Server” are the same as was entered for your PC when setting your static IP in step 5q.
  7. Click Save and Close
  8. Attempt a firmware update
  9. Attempt a transfer or activation
